# 程序化 3D 天体模型

3D 太阳系场景优先使用适合球体映射的等距全球纹理。太阳、水星、金星表面、金星大气、月球表面、火星表面、木星大气、土星大气、土星环、天王星大气和海王星大气已经改为真实影像派生的渲染纹理；不适合球面展开的代表性照片仍只用于信息面板和来源追溯，避免非全球照片造成球面裁切、亮暗方向冲突和动态光照错误。

## 建模原则

- 主体天体优先使用真实影像派生的等距纹理；`CanvasTexture` 仅用于薄雾、光晕、尘埃环等辅助层，不作为行星主表面替代。
- 行星和月球使用 `meshStandardMaterial` / `meshPhysicalMaterial` 接收太阳点光源，亮面和暗面由 3D 光照决定。
- 所有天体主球体保持平滑，不使用噪声位移或陨石坑几何制造凹凸。
- 太阳使用发光几何、柔和日冕和低干扰日珥表现恒星自发光。
- 所有模型保留现有轨道位置、教学压缩半径、真实轴倾角、自转轴、点击选择和标签行为。
- 复杂几何通过 `useMemo` 生成，避免播放轨道动画时重复重建。

## 天体策略

- 太阳：平滑发光球、透明日冕壳、低干扰日珥、太阳黑子 patch。
- 水星：平滑岩质球体，使用暗灰、深褐和灰白色平滑 patch 表现荒凉撞击坑、环形山、古老熔岩平原和曲线断崖。
- 金星：平滑岩质内核，叠加暗黄、焦褐、暗红色火山平原、高地、山脉、峡谷、火山构造、陨击坑以及多层浓厚云壳。
- 地球：高对比海洋球、内置矢量大陆轮廓、低遮挡云层 patch、大气壳。
- 月球：真实 8K 全球月面纹理，呈现冷峻银灰和深灰月海、密集撞击坑、锐利环形山、明亮辐射状溅射纹和月壤粗糙感；程序化层只用于轻微月壤明暗 sheen。
- 火星：真实全球表面纹理，呈现红褐氧化铁沙尘、暗色玄武岩区、水手峡谷、奥林帕斯山、撞击坑和极冠；程序化层只用于薄尘大气和材质质感。
- 木星：真实全球大气纹理，呈现纬向云带、大红斑、涡旋云团和极区气旋；程序化层只用于赤道鼓起、云层 sheen 和薄大气光晕。
- 土星：真实全球大气纹理和真实环纹理，呈现淡金色纬向云带、赤道鼓起和精细环缝；程序化层只用于云层 sheen、薄大气光晕和补充冰粒点。
- 天王星：真实全球大气纹理，呈现柔和淡蓝绿甲烷气溶胶、极区微绿色调、赤道偏蓝对比、高纬风暴云团、暗灰褐低透明度尘埃环、微弱受光侧高光、背光侧散射和横躺自转轴。
- 海王星：真实全球大气纹理，呈现比天王星更浓郁的深蓝甲烷大气、大黑斑、白色甲烷冰晶卷云、超音速急流 sheen、深寒光晕、极弱暗色尘埃环和外侧 Adams 不连续环弧。

## 性能预算

- 主球体几何按模型复杂度控制在 72–192 段范围内。
- 主球体不做位移变形，避免表面凹凸影响观感。
- 土星环粒子约 1800 个点，天王星暗环尘埃约 760 个点，海王星微弱环尘埃约 680 个点。
- 云层、大气和风暴层数保持少量、可识别、可维护。
